Set up Buildr

For security reasons, please don't enable public access to the server until setup is complete.

Step 1: Setup the root account

When you log in to root, you'll use an email just like any other account. Please set the email now. Make sure it is correct, as you can't change it without logging in with it.

Make sure this email is secure, as anyone with access to it can do nearly anything on this website.

This email must be different to your user's email.
